# SilentRoot

A modular PowerShell 7 environment for Windows, built for productivity, automation, and a clean command-line experience.

---

## ✨ Features

- Modular PowerShell profile
- Custom PowerShell modules
- Git helpers
- Browser shortcuts
- Python utilities
- Package manager helpers
- Project launcher
- Automation commands
- Easy to extend
