# SilentRoot

> **A personal PowerShell toolkit.**

SilentRoot is a modular PowerShell environment built around one simple idea:

> Build tools that make everyday work easier.

This repository is a personal hobby project where I experiment, learn, refactor, and gradually build a collection of PowerShell utilities for my own workflow.

It is **not** intended to be a production framework or a commercial product.

---

# Philosophy

Every command in this repository exists for one of three reasons:

- I needed it.
- I wanted to learn something.
- It looked fun to build.

SilentRoot grows one command at a time.

---

# Features

- Modular PowerShell profile
- Built-in Help System
- Markdown-based documentation
- Searchable help index
- Browser shortcuts
- Git helpers
- Python utilities
- Project launcher
- Easy to extend
